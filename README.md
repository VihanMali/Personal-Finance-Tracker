# Personal Finance Manager (Java)

A console-based Java application that helps users manage their personal
finances by tracking income, expenses, savings, and investments.

The application supports both daily and monthly income earners and
automatically allocates money into emergency funds, investments, and
spending categories.

---

## Features

- Supports **Daily Incomer** and **Monthly Incomer**
- Automatic income allocation:
  - Emergency Fund 
  - Investments
  - Spending 
- Investment breakdown:
  - SIPs 
  - Stocks 
  - Gold / Fixed Deposit 
- Daily expense tracking
- Monthly balance summary
- Simple and beginner-friendly console interface

---

## Tech Stack

- Java
- Object-Oriented Programming (OOP)
- Console-based UI

---

## How to Run

1. Clone the repository:
   git clone https://github.com/VihanMali/Personal-Finance-Tracker

2. Navigate to the project folder:
   cd Personal-Finance-Manager-Java

3. Compile the Java files:
   javac *.java

4. Run the program:
   java Main
